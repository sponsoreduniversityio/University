Bachelor of Science in Information Technology (BSIT)

This 3-year online BSIT program is designed to equip students with core IT knowledge including programming, networking, cybersecurity, and systems development.

  • Duration: 3 Years
  • Mode: 100% Online
  • Language: English

Curriculum Structure

Year 1

  • Introduction to IT
  • Computer Programming I
  • Web Technologies
  • Mathematics for Computing
  • Communication Skills

Year 2

  • Database Management
  • Object-Oriented Programming
  • Computer Networks
  • Operating Systems
  • Cybersecurity Basics

Year 3

  • Software Engineering
  • IT Project Management
  • Cloud Computing
  • Capstone Project
  • Internship / Industry Training
